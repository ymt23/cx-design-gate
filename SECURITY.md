# Security Policy

## Sensitive Project Data

Do not commit private product screenshots, proprietary UI requirements, customer data, or project-specific calibration cases to this OSS repository.

Project-local calibration data should stay in the project repository, typically under:

```text
repo/.codex/cx-design-gate/calibration/
```

## Reporting Issues

For now, report security concerns privately to the repository owner before opening a public issue.
