# Generic Pass Case: Primary CTA Screen

Verdict: strong-pass

Pattern:
- One dominant action is visually clear.
- Status content and recent items support the primary action.
- Persistent controls do not compete with content hierarchy.
- Native implementation preserves spacing, readable text, and component boundaries.

Reusable lesson:
A screen with a single dominant CTA and clear content hierarchy is likely to transfer well from concept art to native implementation when the contract keeps component responsibilities explicit.
